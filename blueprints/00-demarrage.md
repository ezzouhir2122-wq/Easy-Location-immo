# Blueprint 00 — Démarrage de Session

## Checklist de Démarrage (30 secondes)

1. **Lire l'état** : `live/state.md` — Que se passait-il à la dernière session ?
2. **Vérifier les tâches** : `live/tasks.md` — Quelles tâches sont prioritaires ?
3. **Focus du moment** : `intel/focus.md` — Quels sont les objectifs de la semaine ?
4. **Décisions passées** : `decisions/ledger.md` — Y a-t-il des choix à respecter ?

## Puis Traiter la Demande

- Si bug → consulter `references/architecture.md` pour le contexte
- Si nouvelle feature → vérifier `live/tasks.md` pour la priorité
- Si question sur la stack → `intel/stack.md`

## En Fin de Session

1. Mettre à jour `live/state.md` (ce qui est fait, ce qui est en cours, blocages)
2. Mettre à jour `live/tasks.md` (cases cochées, nouvelles tâches ajoutées)
3. Logger les décisions importantes dans `decisions/ledger.md`
4. Archiver les fichiers obsolètes dans `archive/`
